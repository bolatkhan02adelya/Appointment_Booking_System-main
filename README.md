# Appointment Booking System project

This is a Java application for managing appointments between providers and clients. We used a medical clinic theme for this version of the project.

## Main features
* Management of Doctors and Patients (CRUD operations)
* Creating appointments between a doctor and a patient
* Connection to PostgreSQL database using JDBC

## Project Structure
* data - DB connection settings
* models - entity classes (Doctor, Patient, Appointment)
* repositories - SQL queries and database logic
* controllers - business logic and app flow

## Technologies
* Java
* PostgreSQL
* JDBC
